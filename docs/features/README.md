# Wiew — Feature Specifications

Each file in this directory is a detailed spec for a major Wiew feature.

## Features

| Spec | Description |
|------|-------------|
| [markdown-preview.md](markdown-preview.md) | Render Markdown in the browser with GitHub-like styling and live reload |
| [ssh-detection.md](ssh-detection.md) | Detect SSH sessions to decide between local browser and public URL |
| [secure-tunneling.md](secure-tunneling.md) | Expose the local server via a temporary token-gated public URL |
| [browser-editing.md](browser-editing.md) | Edit Markdown in the browser; save changes back to disk via WebSocket |
| [cross-platform.md](cross-platform.md) | Build and distribute for Linux, macOS, and Windows |
